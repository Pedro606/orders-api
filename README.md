# Orders API

API simples desenvolvida em Node.js para gerenciamento de pedidos.

## Tecnologias
- Node.js
- Express

## Como executar

Instalar dependências:

npm install

Rodar servidor:

node server.js

## Endpoints

Criar pedido
POST /order

Buscar pedido
GET /order/:id

Listar pedidos
GET /order/list

Atualizar pedido
PUT /order/:id

Deletar pedido
DELETE /order/:id
